OCR Recognition Micro-Service


GOAL

Recognize and extract line text from singal-line image(s).

USAGE

1. Call OCR segmentation micro-service with url: "http://10.5.146.92:8003/recognitionapi".

2. Input:

1) The path of the singal-line image(s), uploaded in form style.
2) Optional. Specified parameter valuses which are listed in the following part, and send them in form style.

3. Output:

A zip file containing all output files related to original image(s). Usually they are named as {imagename}.txt (, {imagename}.prob, and {imagename}.llocs if related parameters have been set).

4. E.g. curl -F "image=@path_to_image1" -F "image=@path_to_image2" -F "probabilities=True" -o name_of_output_file http://10.5.146.92:8003/recognitionapi

PARAMETERS (#11)

Parameter Type Default value Comment
height Integer -1 target line height (overrides recognizer)")
pad Integer 16 extra blank padding to the left and right of text line
nonormalize Boolean False normalize the textual output from the recognizer
llocs Boolean False output LSTM locations for characters
probabilities Boolean False output probabilities for each letter

NOTE

1) The uploaded image(s) must be segmented singal-line image(s).
2) The uploaded image variable name and specified parameters' name must be same with the parameters name in the above table.
