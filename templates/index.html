OCR Recognition Micro-Service


Goal:

Recognize and extract all line text from line images, generate line text files and a over-all file.

Usage:

    1. Call OCR segmentation micro-service with url: "http://10.224.18.222:8003/api/".

    2. Input: the path of the segmented image(s), uploaded in form style. Usually the images are the output of OCR segmentation service similiar "0100??.bin.png", and all of these images belong to a singal origianl image.

    3. Output: a text file containing all of the line text.

    4. E.g.
        For one image: curl -F "file=@path_to_image" -o name_of_output_file http://10.244.18.222:8003/api/
        For two images: curl -F "file1=@path_to_image1" -F "file2=@path_to_image2" -o name_of_output_file http://10.244.18.222:8003/api/

Note:

    The image(s) must be the segmented image(s), or individual line text image(s).
